# course_schedule_app

一个用于课程表管理与查看的应用项目。

## 项目简介

`course_schedule_app` 旨在提供一个清晰、易用、可扩展的课程表管理工具，用来记录、查看和维护个人或班级课程安排。

这个仓库目前用于：
- 存放项目源码
- 管理文档与资源
- 记录后续迭代计划

## 功能特性

- 课程表查看
- 课程信息管理
- 上课时间与节次展示
- 后续可扩展提醒、统计、同步等功能
